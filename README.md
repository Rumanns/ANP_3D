### ANP\_3D\_PATRIMONIAL



##### Objetivos:

1. Representar toda estrutura física da ANP em 3D;
2. Mostrar onde cada bem patrimonial se encontra setor;
3. Localizar cada bem através do seu número de série;
4. Observar as características de cada bem.



##### Especificações:

Blender 3D ----- vs5.0

Python ----- vs3.0
